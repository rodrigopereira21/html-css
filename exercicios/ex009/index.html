Listas

Listas ordenadas

Para criar um ordered list (lista ordenada) vamos usar a tag <ol> para delimitar a lista e <li>(list item, listar item) para indetinficar cada item da lista.

  1. item
  2. item
  3. item
  4. item
  5. item

Listas não ordenadas

Para criar uma lista não ordenada(unordered list) vamos usar a tag <ul> para delimitar a lista e <li> para criar cada um de seus itens internos.

  • item 3
  • item 1
  • item 5
  • item 2

Misturando as coisas

Podemos também criar listas mistas, configurando listas dentro de outras listas. Veja:

  • Nitendinho
    1. Mario Bros
      • Super Mario Bros 3
      • Mario: Lost Levels
  • Super Nitendo
    1. Super Mario World
    2. Donkey Kong
      • Donkey Kong Country III
      • Diddy's Kong Quest
  • Playstation 1
    1. Final Fantasy
      • Final Fantasy VII
      • Final Fantasy IX
    2. Castlevania
      • Simphony of the Night
